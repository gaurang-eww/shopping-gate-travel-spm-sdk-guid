# Shopping Gate Travel SDK — iOS SPM Guide

Published documentation site for integrating the Shopping Gate Travel SDK
into a native iOS app via Swift Package Manager.

- Source content: [`IOS_SPM_INTEGRATION_GUIDE.md`](./IOS_SPM_INTEGRATION_GUIDE.md)
- Rendered via [Docsify](https://docsify.js.org) — `index.html` loads the
  markdown file directly, no build step.
- Published at GitHub Pages once enabled: repo **Settings → Pages → Source:
  Deploy from a branch → `main` → `/ (root)`**. No build step or Actions
  workflow needed — it's served as plain static files.

To update the guide, edit `IOS_SPM_INTEGRATION_GUIDE.md` and push — no
rebuild needed.
